PROJECT CONTEXT:
This license applies to the "Start/Stop VM Azure Function App" project, 
an automated serverless solution for managing Azure Virtual Machine lifecycles 
based on predefined schedules. The project demonstrates cloud automation, 
cost optimization, and serverless architecture patterns using Azure Functions, 
Azure Compute Management, and timer-triggered operations.

Developed as part of a comprehensive cloud functions portfolio showcasing 
Azure serverless capabilities, infrastructure automation, and DevOps practices.
